Change Log

v 0.2 [Bamboo]

The first public relase of vx.io. The following features/fixes were added in vx.io 0.2:

  • Additional syntax highlighting support (Java Script, PHP, CSS)
  • Added Find/Replace Functionality.
  • Added Console Support and Directory Tree View.
  • Many other Misc Bug Fixes.

See README.txt for more information.

v 0.1 [Apricot]

The first internal "release" of vx.io(called vxPad at the time). It's considerde na Alpha release. The following features/fixes were added in vx.io 0.1:

  • Basic Syntax Highlighting (C, C++, C# HLSL, Python and HTML).
  • Basic File IO Operations.
  • Toggle EOL View, ALL CAPS, lower caps. Able to Change EOL Format for different platforms.
  • Implementation of wxAUI classes to provide GUI Layout.

See README.txt for more information.
